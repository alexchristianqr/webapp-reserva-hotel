# Especificaciones de features

Cada feature vive en su propia carpeta numerada:

```
specs/
└── NNN-nombre-feature/
    ├── spec.md     ← Fase 1: QUÉ y POR QUÉ  (de /specify)
    ├── plan.md     ← Fase 2: CÓMO técnico   (de /plan)
    └── tasks.md    ← Fase 3: tareas         (de /tasks)
```

La numeración es incremental de tres dígitos: `001-`, `002-`, ...

Ver el flujo completo en `.claude/sdd/workflow.md`.
